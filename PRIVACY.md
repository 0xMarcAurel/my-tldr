# Privacy Policy

This application ("my-tldr") processes publicly available Reddit post content to generate summaries using an external AI service (Google Gemini).

## Data Usage

- The app reads post titles and body content from Reddit.
- For link posts, the app may attempt to fetch publicly available webpage content.
- Content may be sent to a third-party API (Google Gemini) for summarization.

## Data Storage

- The app does not store personal data.
- No user data is retained after processing.

## Third-Party Services

- Google Gemini API is used to generate summaries.
- Their data handling policies apply to any data sent for processing.

## User Control

- This tool is only accessible to subreddit moderators.
- Users should verify all generated summaries independently.

## Changes

This policy may be updated over time.
